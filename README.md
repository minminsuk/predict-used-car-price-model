# AI 기반 중고차 가격 예측 웹 서비스

## 프로젝트 개요
사용자가 차량 정보를 입력하면 머신러닝 모델을 활용하여
중고차 가격을 예측하는 웹 서비스 개발

## 주요 기능

- 차량 정보 입력
- AI 기반 가격 예측
- 모델 예측 결과 출력

# 프로젝트 분석 (Analysis)

## 프로젝트 목적

중고차 가격은 차량 모델, 연식, 주행거리 등 다양한 요인에 의해
결정되며 일반 사용자가 적정 가격을 판단하기 어렵다.

본 프로젝트는 머신러닝 모델을 활용하여 차량 정보를 입력하면
예상 중고차 가격을 제공하는 웹 서비스를 개발하는 것을 목표로 한다.

## 예상 사용자

- 중고차 구매자
- 중고차 판매자
- 자동차 시장 데이터에 관심 있는 사용자

## 사용 데이터

입력 데이터

- 차량 모델
- 연식
- 주행거리
- 연료 종류
- 트림

출력 데이터

- 예측 중고차 가격

## 프론트엔드

역할
- 사용자 입력 화면 제공
- 차량 정보 입력
- 예측 결과 출력

## 백엔드

역할
- REST API 제공
- 머신러닝 모델 실행
- 예측 결과 반환

## (머신러닝/ 딥러닝) 모델

사용 모델

- RandomForest
- XGBoost
- Deep Neural Network

모델 역할

차량 데이터(연식, 주행거리 등)를 입력받아
중고차 가격을 예측

## 기대 효과

- 중고차 가격 판단에 도움 제공
- 데이터 기반 의사결정 지원
- AI 기술 활용 웹 서비스 구현
